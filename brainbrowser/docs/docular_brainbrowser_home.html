BrainBrowser API Documentation

BrainBrowser is a set of web-based, 3D visualization tools primarily targetting neuroimaging. Using open web-standard technologies, such as WebGL and HTML5, it allows for real time manipulation and analysis of 3D imaging data whether it be precalculated maps, such as the MACACC Dataset (Mapping Anatomical Correlations Across Cerebral Cortex), or models provided by the user in MNI object, Wavefront object or FreeSurfer ASCII surface file formats, and data in one of the many currently supported formats (Minc, Nifti, object files, plain text). BrainBrowser also provides a Volume Viewer for exploring 3D MINC volume data.
